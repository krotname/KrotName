# Featured Project Rotation

Use this note when deciding which repositories should stay in the first profile screen.

## Maintenance Checks

- Prefer maintained repositories with visible tests or releases.
- Balance Java, desktop tools, automation, and infrastructure examples.
- Retire projects that no longer represent current work.

## Validation

Review the profile README after applying this note. The profile should stay concise, bilingual where needed, and aligned with the current public repository set.
