# Quality Signal Copy Rules

Use this note when describing tests, releases, supply-chain work, or security hygiene.

## Maintenance Checks

- Prefer concrete repository practices over broad claims.
- Keep quality wording current with the actual workflows.
- Do not imply coverage that a repository does not have.

## Validation

Review the profile README after applying this note. The profile should stay concise, bilingual where needed, and aligned with the current public repository set.
