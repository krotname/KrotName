# Security Link Review

Use this note when security policy links or vulnerability-reporting language affect profile claims.

## Maintenance Checks

- Check that highlighted repositories have SECURITY.md when security posture is mentioned.
- Prefer private vulnerability reporting links where available.
- Avoid exposing private operational details from security notes.

## Validation

Review the profile README after applying this note. The profile should stay concise, bilingual where needed, and aligned with the current public repository set.
