# Public Repository Hygiene Notes

Use this note when the profile points readers toward public repositories.

## Maintenance Checks

- Highlight repositories with clear README files and reproducible commands.
- Prefer projects with visible security and contribution files.
- Avoid sending profile readers to stale experiments.

## Validation

Review the profile README after applying this note. The profile should stay concise, bilingual where needed, and aligned with the current public repository set.
