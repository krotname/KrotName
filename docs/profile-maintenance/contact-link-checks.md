# Contact Link Checks

Use this note when contact links or public identity links change.

## Maintenance Checks

- Verify the personal site resolves over the intended canonical URL.
- Keep contact links short and public-safe.
- Avoid adding direct private contact details to the profile README.

## Validation

Review the profile README after applying this note. The profile should stay concise, bilingual where needed, and aligned with the current public repository set.
