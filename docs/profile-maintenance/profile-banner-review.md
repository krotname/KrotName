# Profile Banner Review

Use this note before updating banner assets or their alt text.

## Maintenance Checks

- Keep the banner readable at normal GitHub profile width.
- Update alt text when the visual message changes.
- Avoid large binary churn unless the asset visibly improves the profile.

## Validation

Review the profile README after applying this note. The profile should stay concise, bilingual where needed, and aligned with the current public repository set.
