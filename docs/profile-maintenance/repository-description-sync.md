# Repository Description Sync

Use this note when GitHub repository descriptions and profile summaries diverge.

## Maintenance Checks

- Keep profile project summaries aligned with repository metadata.
- Use the profile README for reader-facing context, not full release notes.
- Refresh stale descriptions after major repo purpose changes.

## Validation

Review the profile README after applying this note. The profile should stay concise, bilingual where needed, and aligned with the current public repository set.
