# Repository Catalog Maintenance

Use this note when the selected project list changes or a repository description is refreshed.

## Maintenance Checks

- Keep project names matched to their GitHub repository names.
- Prefer stable project summaries over temporary implementation details.
- Remove archived or forked repositories from highlighted lists unless they are intentionally shown.

## Validation

Review the profile README after applying this note. The profile should stay concise, bilingual where needed, and aligned with the current public repository set.
