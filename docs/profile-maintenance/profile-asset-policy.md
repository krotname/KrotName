# Profile Asset Policy

Use this note when changing SVG, badge, or image assets.

## Maintenance Checks

- Keep assets small enough for quick profile loading.
- Review SVG text and links before publishing.
- Avoid decorative churn that does not improve the profile signal.

## Validation

Review the profile README after applying this note. The profile should stay concise, bilingual where needed, and aligned with the current public repository set.
