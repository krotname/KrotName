# Tooling Project Balance

Use this note when balancing editor tools, backend services, and infrastructure projects.

## Maintenance Checks

- Keep at least one desktop/editor example visible when it is actively maintained.
- Avoid overloading the profile with similar backend projects.
- Prefer a mix that shows breadth without looking unfocused.

## Validation

Review the profile README after applying this note. The profile should stay concise, bilingual where needed, and aligned with the current public repository set.
