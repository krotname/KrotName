# Automation Copy Review

Use this note when describing automation projects in public profile copy.

## Maintenance Checks

- Describe the user-facing task the automation performs.
- Avoid exposing private machine names or credentials.
- Keep implementation details in project repositories, not the profile summary.

## Validation

Review the profile README after applying this note. The profile should stay concise, bilingual where needed, and aligned with the current public repository set.
